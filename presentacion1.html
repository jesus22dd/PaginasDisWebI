ACTIVIDAD III - CSS

15 de Octubre, 2024

CSS, o Cascading Style Sheets, es un lenguaje utilizado para describir la presentación de un documento HTML o XML. Permite controlar la apariencia de los elementos en una página web desde los colores hasta la disposición, fuentes y espaciado.

Objetivo General

Demostrar la relevancia del CSS en el diseño web, subrayando su efectividad y rapidez en comparación con HTML.

Objetivos Específicos

  • Definir el CSS y detallar su origen así como su evolución en el ámbito del desarrollo web.
  • Ilustrar diferentes sintaxis de CSS a través de ejemplos prácticos.
  • Crear un diseño web que evidencie las capacidades del CSS.
  • Analizar cómo el CSS influye en la experiencia del usuario.
  • Identificar las mejores prácticas para la implementación del CSS.

CSS (Cascading Style Sheets) es un lenguaje de diseño que define la presentación y el formato de un documento HTML o XML. Permite a los desarrolladores separar el contenido estructural de su apariencia visual.

  • Separación de Contenido y Presentación: Permite modificar el diseño sin afectar la estructura del contenido.
  • Diseño Responsivo: CSS permite que los sitios se adapten a distintos tamaños de pantalla.
  • Soporte Multiplataforma: CSS es soportado por todos los navegadores modernos.
  • Mejora en la Mantenibilidad: Los cambios se aplican fácilmente a través de hojas de estilo externas.

CSS es una herramienta esencial en el diseño web moderno. Su capacidad para controlar la apariencia de las páginas de manera eficiente y flexible lo convierte en una tecnología fundamental para crear sitios visualmente atractivos y funcionales.
